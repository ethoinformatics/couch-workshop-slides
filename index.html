CouchDB and PouchDB

NoSQL, web-oreinted databases

What is CouchDB?

  • Open source document oriented database
  • All operations are accessed through HTTP
  • All documents are JSON
  • Written in erlang
  • Very simple replication

What is PouchDB?

  • You can't run an erlang web-server in a web-browser, so PouchDB!
  • PouchDB re-implements the same replication protocol as CouchDB, so syncing between Pouch and Couch is a breeze
  • PouchDB creates an in-browser database using the best available technology(IndexedDB, WebSQL)

Why this is a big deal

  • Our mobile app is just a web-page running in a frameless web-browser
  • Researchers are typically offline while collecting data

CRUD

Familiar database operations

  • Create - INSERT
  • Read - SELECT
  • Update - UPDATE
  • Delete - DELETE

CRUD vs The Web

  • Create - PUT
  • Read - GET
  • Update - POST
  • Delete - DELETE
